Features

  • Real-time event logging with timestamps
  • Automated daily check-ins (08:00 and 20:00)
  • Event filtering by date range and text
  • Print functionality for reports
  • Selective event deletion

Morning Check-in (08:00)

  • Weather conditions
  • Property status
  • Vehicle status
  • Health check-in
  • Upcoming events

Evening Check-in (20:00)

  • Mood tracking
  • Future event planning
  • Daily summary

Usage

Enter events in the input field at the top and press Enter or click 'Log Event'. Use filters to search through past events. Select events with checkboxes for deletion.
